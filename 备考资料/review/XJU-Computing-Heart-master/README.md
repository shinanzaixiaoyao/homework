## 🫀目的？

​	意为新疆大学信息学院学生提供学习便利，免去寻找往年试题烦恼。亦为其他专业同学学习信息院专业课程提供便利条件。

### 💻展望

​	计划发展成为新疆大学信息工程学院的飞跃手册，提供 ⚠️【便利的 公开的 清晰的】⚠️【学业上，职业上，生活上，思想上】的⚠️「参考」
